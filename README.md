#Study Platform – Frontend

Projeto frontend criado com o objetivo de praticar e consolidar conhecimentos em desenvolvimento frontend, consumindo uma API backend desenvolvida em Java com Spring Boot.

Este frontend é responsável pela interface do usuário e pela comunicação com o backend do projeto.

#Backend
A API utilizada por este frontend está disponível em:
https://github.com/profissionalArodrigues/studyplatform

#Tecnologias utilizadas

- HTML
- CSS
- JavaScript

#Objetivo do projeto
Projeto de estudo voltado para:

- Consumo de APIs REST
- Estruturação de aplicações frontend
- Integração com backend
- Boas práticas de desenvolvimento frontend
